# Dataset Design

## Project Purpose

This project studies how geopolitical risk affects Taiwan financial markets.

The dataset is organized around event dates. Each event is matched with market data before, during, and after the event.

## Core Datasets

### `data/events.csv`

Purpose:
Stores geopolitical and strategic events related to Taiwan.

Columns:

| Column | Description |
| --- | --- |
| `date` | Date of the event. |
| `event_name` | Short event name. |
| `event_type` | Event category, such as diplomatic, political, or investment. |
| `risk_level` | Estimated geopolitical risk score from 1 to 5. |
| `strategic_value` | Estimated strategic importance score from 1 to 5. |
| `source` | Source note or URL for the event. |

Example rows:

| date | event_name | event_type | risk_level | strategic_value | source |
| --- | --- | --- | ---: | ---: | --- |
| 2022-08-02 | Pelosi Visit | diplomatic | 5 | 4 | ... |
| 2024-05-20 | Lai Inauguration | political | 4 | 3 | ... |
| 2025-05-19 | Nvidia AI Factory Announcement | investment | 1 | 5 | ... |

### `data/market.csv`

Purpose:
Stores Taiwan financial market data by date.

Columns:

| Column | Description |
| --- | --- |
| `date` | Market observation date. |
| `taiex` | TAIEX index value. |
| `usd_twd` | USD/TWD exchange rate. |
| `tsmc` | TSMC stock price. |

Example rows:

| date | taiex | usd_twd | tsmc |
| --- | ---: | ---: | ---: |
| 2022-08-01 | 14981 | 29.90 | 500 |
| 2022-08-02 | 14821 | 30.05 | 490 |
| 2022-08-03 | 14567 | 30.30 | 480 |

### `data/event_scores.csv`

Purpose:
Stores coded risk and strategic-value scores with short reasoning.

Columns:

| Column | Description |
| --- | --- |
| `date` | Date of the event. |
| `event_name` | Event name. |
| `risk_level` | Estimated geopolitical risk score from 1 to 5. |
| `strategic_value` | Estimated strategic importance score from 1 to 5. |
| `reasoning` | Short justification for the assigned scores. |

Example rows:

| date | event_name | risk_level | strategic_value | reasoning |
| --- | --- | ---: | ---: | --- |
| 2022-08-02 | Pelosi Visit | 4 | 4 | Triggered major PLA response and highlighted Taiwan's strategic importance. |

## Derived Datasets

### `output/event_summary.csv`

Purpose:
Combines each event with same-day market values.

Columns:

| Column | Description |
| --- | --- |
| `date` | Event date. |
| `event_name` | Event name. |
| `event_type` | Event category. |
| `risk_level` | Estimated geopolitical risk score from 1 to 5. |
| `strategic_value` | Estimated strategic importance score from 1 to 5. |
| `taiex` | TAIEX value on the event date. |
| `usd_twd` | USD/TWD value on the event date. |
| `tsmc` | TSMC value on the event date. |

Created by:
`event_analysis.py`

### `output/event_impact.csv`

Purpose:
Measures short-run market movement around each event.

Columns:

| Column | Description |
| --- | --- |
| `event_name` | Event name. |
| `event_type` | Event category. |
| `risk_level` | Estimated geopolitical risk score from 1 to 5. |
| `strategic_value` | Estimated strategic importance score from 1 to 5. |
| `taiex_before` | TAIEX value one day before the event. |
| `taiex_event_day` | TAIEX value on the event date. |
| `taiex_after` | TAIEX value one day after the event. |
| `taiex_change` | Percentage change in TAIEX from before to after. |
| `usd_twd_before` | USD/TWD value one day before the event. |
| `usd_twd_event_day` | USD/TWD value on the event date. |
| `usd_twd_after` | USD/TWD value one day after the event. |
| `usd_twd_change` | Percentage change in USD/TWD from before to after. |
| `tsmc_before` | TSMC value one day before the event. |
| `tsmc_event_day` | TSMC value on the event date. |
| `tsmc_after` | TSMC value one day after the event. |
| `tsmc_change` | Percentage change in TSMC from before to after. |

Created by:
`event_impact.py`

## Variable Design

| Variable Type | Variables |
| --- | --- |
| Independent variables | Geopolitical risk, strategic importance. |
| Dependent variables | TAIEX return, USD/TWD change, TSMC change, investment announcement count. |
| Event identifier | `date`, `event_name`. |
| Market indicators | `taiex`, `usd_twd`, `tsmc`. |

## Event Coding Rules

Detailed rules are stored in `event_coding_rules.md`.

| Field | Scale | Meaning |
| --- | --- | --- |
| `risk_level` | 1 to 5 | Estimated geopolitical risk generated by the event. |
| `strategic_value` | 1 to 5 | Degree to which the event highlights Taiwan's strategic role in technology, security, or supply chains. |

## Recommended Expansion

Future versions of the dataset can add:

1. Event category, such as crisis, election, sanction, investment, or policy announcement.
2. Event source URL.
3. Event severity score.
4. Longer market windows, such as 3-day, 5-day, and 10-day changes.
5. Real historical market prices for all event windows.
