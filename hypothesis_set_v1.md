# Hypothesis Set V1

## Purpose

This hypothesis set follows the parsimonious theory:

```text
Grey-Zone Sovereignty -> Repeated Shocks -> Investor Adaptation -> Market Reaction
```

The hypotheses are designed for future testing. They should not be presented as established findings.

## Primary Hypothesis

### H1: Adaptation Hypothesis

As Taiwan-related geopolitical shocks become more repeated, their negative abnormal-return effects weaken.

Expected pattern:

```text
Higher event_repetition -> less negative abnormal returns
```

Main dependent variables:

- TWSE CAR [-7,+7]
- TSMC CAR [-7,+7]
- average TWSE/TSMC CAR [-7,+7]

## Secondary Hypotheses

### H2: Threat Baseline Hypothesis

Events interpreted as threats are associated with more negative abnormal returns than non-threat events.

Expected pattern:

```text
Risk events -> negative average abnormal returns
```

This is the baseline expectation. It should be tested before evaluating adaptation.

### H3: Opportunity Hypothesis

Events interpreted as strategic-importance signals are associated with more positive abnormal returns.

Expected pattern:

```text
Strategic_Importance events -> positive average abnormal returns
```

This hypothesis applies mainly to semiconductor, CHIPS Act, TSMC, and supply-chain events.

### H4: Surprise Conditioning Hypothesis

More surprising events are associated with larger absolute abnormal returns.

Expected pattern:

```text
Higher surprise_level -> larger absolute abnormal returns
```

This hypothesis concerns reaction size, not necessarily direction.

## Scope Conditions

The adaptation hypothesis should apply most directly to repeated event families, especially:

- Taiwan Strait military demonstrations
- PLA exercises after political triggers
- repeated sovereignty-related signaling

The opportunity hypothesis should apply most directly to:

- TSMC investment events
- CHIPS Act events
- semiconductor supply-chain restructuring
- technology-competition events

The threat hypothesis should apply most directly to:

- novel military escalation
- coercive economic measures
- diplomatic shocks with low precedent

## Minimum Testable Model

The minimum empirical model should include:

| variable | role |
| --- | --- |
| abnormal return | dependent variable |
| event_repetition | primary independent variable |
| mechanism | secondary independent variable |
| surprise_level | conditioning variable |
| event_category | control or subgroup variable |

## Expected Contribution of the Hypotheses

The hypotheses move the project away from a simple severity model:

```text
Higher risk -> more negative return
```

and toward a repetition model:

```text
Repeated risk -> adaptation -> weaker negative return
```

This is the minimum theory needed to explain why some Taiwan-related geopolitical shocks fail to generate persistent market damage despite elevated objective risk.
