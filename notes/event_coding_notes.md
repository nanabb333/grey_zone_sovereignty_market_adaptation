# Event Coding Notes

Events are classified into two main categories:

- `Risk`: events primarily associated with military pressure, diplomatic escalation, sanctions threats, coercion, or direct Taiwan Strait instability.
- `Strategic_Importance`: events primarily associated with Taiwan's semiconductor centrality, allied industrial strategy, investment commitments, or policy developments that reinforce Taiwan's geoeconomic importance.

Coding requires judgment because some events may contain both threat and opportunity. In those cases, the classification reflects the dominant interpretation used in the empirical comparison.

Variables used in the broader project include military risk, diplomatic risk, sanctions risk, semiconductor relevance, AI relevance, security relevance, surprise level, event repetition, and coding confidence.
