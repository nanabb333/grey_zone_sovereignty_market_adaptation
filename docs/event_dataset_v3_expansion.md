# Event Dataset V3 Expansion Review

> Academic Research Notice:
> This document creates a review pipeline for candidate events. It does not insert any event into the validated event-study dataset.

## Purpose

Event Dataset V3 should expand the empirical foundation of the Taiwan geopolitical risk project while preserving a clear boundary between candidate discovery and validated event-study inclusion. The candidates below were reviewed against four practical criteria:

- The event is real and historically dated.
- The event has identifiable public actors and a plausible investor-interpretation channel.
- The event is relevant to Taiwan, semiconductors, export controls, strategic technology, industrial policy, supply-chain security, or diplomatic competition.
- The event is not already represented in the validated V2 event sample unless the candidate adds distinct information.

## Expansion Implication

Using `data/events_v2.csv` as the conservative validated baseline, the 20 Include candidates would increase the review-ready dataset from 18 to 38 events. If all three Review Further candidates pass evidence and confounding checks, the upper-bound V3 candidate pool would be 41 events.

The expansion would materially improve coverage of export controls, semiconductor state support, Chinese countermeasures, and diplomatic-recognition competition. It would not substantially add new military-exercise events because the strongest military candidates from the shortlist are already represented in V2.
