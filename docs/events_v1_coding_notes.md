# Events V1 Coding Notes

## Purpose

This file documents the first draft coding decisions for `data/events_v1.csv`.

The scores are a working draft for manual review. They are not final research claims.

## Coding Scale

| Score | Meaning |
| ---: | --- |
| 1 | Very low |
| 2 | Low |
| 3 | Medium |
| 4 | High |
| 5 | Very high |

## Review Rule

Every row should be manually reviewed before final analysis. If a source does not directly support a specific score, revise the score or add a better source.
