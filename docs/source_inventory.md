# Source Inventory

This document categorizes all major sources used in the Taiwan Grey-Zone Sovereignty project.

---

# Tier A — Core Theory Sources

These sources form the project's theoretical foundation.

| Source                      | Concept                 | Project Role                                                    |
| --------------------------- | ----------------------- | --------------------------------------------------------------- |
| Wendt (1992)                | Recognition             | Explains why sovereignty depends on recognition by other states |
| Shen (2000)                 | Statehood and Taiwan    | Core Taiwan sovereignty literature                              |
| Chiang (2025)               | Taiwan sovereignty      | Modern sovereignty discussion                                   |
| Mearsheimer (2025)          | Great-power competition | Explains structural constraints on Taiwan                       |
| Baldwin (2020)              | Economic Statecraft     | Explains economic tools as instruments of power                 |
| Eichengreen (2011)          | Dollar Power            | Explains U.S. monetary dominance                                |
| Helleiner (2008)            | Monetary Power          | Supports dollar hierarchy arguments                             |
| Caldara & Iacoviello (2022) | Geopolitical Risk       | Measurement framework for geopolitical risk                     |
| Wigell et al. (2018)        | Geoeconomics            | Connects economics and security                                 |
