# Portfolio Impact Assessment

> Academic Research Notice:
> This assessment explains the value of a richer event dataset for academic and portfolio presentation. It does not add product features or report new empirical results.

## Summary Comparison

| Dimension | Current version | Projected V3 version |
| --- | --- | --- |
| Academic value | Small but transparent event-study sample with strong Taiwan military-risk focus. | Broader geopolitical competition dataset with richer policy, technology, and diplomatic coverage. |
| Analytics value | Useful for initial event-study demonstration and adaptation discussion. | More useful for comparing event families, theory variables, repeated events, and policy escalation. |
| Portfolio value | Shows ability to build a reproducible research pipeline. | Shows ability to scale a research dataset while preserving validation, auditability, and methodological discipline. |
| Recruiter value | Demonstrates Python/data workflow and geopolitical finance framing. | Demonstrates research operations, dataset governance, domain expertise, and analytical product judgment. |

## Academic Value

The current dataset is academically useful because it is transparent and cautious. Its main limitation is size and concentration: military-signaling events dominate the sample, while export controls, state support, and diplomatic-recognition competition are underrepresented.

Projected V3 improves academic value by broadening the empirical foundation. It supports stronger claims about geopolitical competition as a multi-domain phenomenon rather than only a Taiwan military-risk sequence.

## Analytics Value

The current version supports basic event windows, abnormal-return summaries, and mechanism-level interpretation. Projected V3 would support richer descriptive comparisons across policy domains. It would also improve the usefulness of theory variables because more events would carry meaningful `strategic_importance_level` and `state_support_signal` coding.

## Portfolio Value

The current version already demonstrates a coherent research pipeline. Projected V3 would demonstrate a higher-value skill: scaling a dataset without sacrificing research integrity. The review pipeline, inclusion protocol, and audit documents show that expansion is governed rather than ad hoc.

## Recruiter Value

For research, policy, finance, risk, or data roles, the projected V3 version is stronger because it shows:

- domain-specific event discovery
- source discipline
- schema governance
- theory-to-data translation
- explicit distinction between candidates and validated observations
- restraint around not fabricating CAR results or synthetic events

## Main Tradeoff

The expansion increases analytical potential but also increases validation burden. The portfolio value is strongest if the project continues to show the boundary between discovered candidates, validated events, and computed results.

## Bottom Line

Projected V3 would make the project look less like a small proof of concept and more like a serious geopolitical competition research dataset. The most important portfolio improvement is not the larger event count by itself; it is the visible research governance around that larger count.
